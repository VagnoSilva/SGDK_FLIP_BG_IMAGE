Essa função permite com que o usuário consiga flipar horizontaalmente
uma imagem sem precisar ter que redesenha-la, a função flipa os tiles 
da imagem e reordena as colunas para que a imagem fique desenhada na orientação correta. 


exemplo de uso:


drawImageFlippedH(BG_A, &bga_image, TILE_USER_INDEX, 0, 0, PAL2);
